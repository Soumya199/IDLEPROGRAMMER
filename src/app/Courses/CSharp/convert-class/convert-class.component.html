Convert Class in C#

What is convert class?

  • Convert class provides static methods to one base type to another base type.
  • The supported base data types such as Boolean, Char, SByte, Int16, Int32, Int64, UInt16, UInt32, UInt64, Single, Double, Decimal, DateTime and String etc.

Parse() :

  • Parse() is used to convert a string into a primitive data type
  • Parse() method throws exception if the string cannot be converted to the specified type
  • Parse() method ignores white space at the beginning and at the end of the string, but all other characters must be the characters that form the appropriate numeric type Any whitespace within the characters that form the number cause an error
  • Syntax:
    • Datatype var_name = Datatype.Parse(String);
  • Example:
    • int number = int.Parse(“55”);

TryParse() :

  • The job of Parse() and TryParse() is same it is also used to convert a string into a primitive data type
  • TryParse() method returns a Boolean to show whether the parsing worked or not.
  • TryParse() method ignores white space at the beginning and at the end of the string, but all other characters must be the characters that form the appropriate numeric type Any whitespace within the characters that form the number cause an error
  • The second keyword is used for the output variable, the ‘out’ keyword assigns the value to the variable
  • Syntax:
    • bool var_name = Datatype.TryParse(String, out result);
  • Example:
    • int year ;
    • bool validYear = int.TryParse(“2020”, out year);
