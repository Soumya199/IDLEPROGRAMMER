Var Vs Dynamic

Use of Var in C#

  • Var keyword was introduced in C# 3.0.
  • Var is a statically typed variable whose value is determined at compile time, That means when you use var keywod in your program at compile time the type of that variable is autodetected. So as it's type is auto detected at compile time you must have to initialize it at the time of declaration otherwise it will give you a compile time error – “Implicitly-typed local variables must be initialized”.
  • It does not allow you to change the value from one type to another once it is declared. for example we can say that if we have declared var varInt=1 we will not be able to change it like varInt="ProgramingWizard" ,if we do that that will show an error like this "Cannot implicitly convert type string to int"

Let's see an Example : 

            
                using System;
                namespace ProgramingWizard
                {{ '{' }}
                
                    public class Program
                    {{ '{' }} 
                    
                        public static void Main(string[] args)
                        {{ '{' }}
                        var varInt = 100;
                        Console.WriteLine(varInt);
                        {{ '}' }}
                    {{ '}' }}
                {{ '}' }}
        
            Output : 100

Use of Dynamic in C#

  • Dynamic keywod was introduced in C# 4.0.
  • Dynamic is a dynamically typed variable whose value is determined at run time, That means when you use dynamic keyword in your program at run time the type is autodetected. As it's type is detected at run time it does not require initialization at the time of declaration.
  • It is opposite to Var keyword. It allows you to change the variable type and not only it will compile but also it will run with out giving any error.
Let's see an Example : 
            
                using System;
                namespace ProgramingWizard
                {{ '{' }}
                
                    public class Program
                    {{ '{' }}
                    public static void Main(string[] args)
                    
                        {{ '{' }}
                        dynamic intvalue=100;
                        Console.WriteLine(intvalue);
                        intvalue="programingWizard";
                        Console.WriteLine(intvalue);
                        {{ '}' }}
                    {{ '}' }}
                {{ '}' }}
            Output : 100
            programingWizard

I have initialized the variable to show you how it works if you do not initialize it at the time of declaration it will not give you any error.
