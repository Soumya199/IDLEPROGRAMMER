Operators in C#

  • Operators are the symbols that have special meaning and used to perform specific mathematical or logical manipulations.
  • Operators are used to perform some operations on variables and values like addition, subtraction, multiplication, etc.
    • Example:
    a + b
  • In this example + is considered as operator and both a & b are called operands.
There are different Operators are there let's discuss some of that.
  1. Arithmetic Operators
  2. Relational Operators
  3. Logical Operators
  4. Bitwise Operators
  5. Assignment Operators
Arithmetic Operators
  • Arithmetic operators are used to perform mathematical operations.
Operator Operator Name Example Description
+ Addition Operator a + b This operator is used to add two values
- Subtraction Operator a - b This operator is used to subtraction one value from another
* Multiplication Operator a * b This operator is used to multiply two values
/ Division Operator a / b This operator is used to divide one value by other
% Modulus Operator a / b This operator is used to return the division remainder
Example :- 
        
            using System;
            class IdleProgrammer
            {{'{'}}
                static void Main() 
                {{'{'}}
                    int a=10;
                    int b=5;
                    int c;
                    c=a+b;
                    Console.WriteLine("Addition of a and b is :"+c);
                    c=a-b;
                    Console.WriteLine("Subtraction of a and b is :"+c);
                    c=a*b;
                    Console.WriteLine("Multiplication of a and b is :"+c);
                    c=a/b;
                    Console.WriteLine("Division of a by b is :"+c);
                    c=a%b;
                    Console.WriteLine("Division remainder a by b is :"+c);
                {{'}'}}
            {{'}'}}

            Output:
                Subtraction of a and b is :5                                                                                                           
                Multiplication of a and b is :50                                                                                                       
                Division of a by b is :2                                                                                                               
                Division remainder a by b is :0

Relational Operators

  • Relational operators are used to check the relation between two operands and this always returns a boolean value (true/false).
Operator Operator Name Example(x=10,y=5) Description
== Equal to x == y returns false This operator is used to compare two values.
> Greater than x > y returns true This operator checks whether left operand greater than the right operand or not and returns true if it is satisfied.
< Less than x < y returns false This operator checks whether left operand less than right operand or not and return true if it is satisfied.
>= Greater than or Equal to x >= y returns true This operator checks whether left operand greater than or equal to right operand or not and return true if it is satisfied.
<= Less than or Equal to x >= y returns false This operator checks It compares whether left operand less than or equal to right operand or not and return true if it is satisfied.
!= Not Equal to x != y returns false This operator checks wheather two operands are equal or not equal if not equal it will return true.
Example :- 
        
            using System;

            namespace IdleProgrammer
            {{'{'}}
                class Program
                {{'{'}}
                    static void Main(string[] args)
                    {{'{'}}
                        bool result;
                        int x = 10, y = 5;
                        result = (x == y);
                        Console.WriteLine("is x equals y ?: " + result);
                        result = (x > y);
                        Console.WriteLine("is x greatet than y?: " + result);
                        result = (x < y);
                        Console.WriteLine("is y greatet than x?: " + result);
                        result = (x >= y);
                        Console.WriteLine("is x greater than or equals y?: "+ result);
                        result = (x <= y);
                        Console.WriteLine("is x less than or equals y?: "+ result);
                        result = (x != y);
                        Console.WriteLine("is x not equals y? : " + result);
                    {{'}'}}
                {{'}'}}
            {{'}'}}
            Output:
                    is x equals y ?: False
                    is x greatet than y?: True
                    is y greatet than x?: False
                    is x greater than or equals y?: True
                    is x less than or equals y?: False
                    is x not equals y? : True

Logical Operators

  • Logical operators are used to varify two or more conditions.
Operator Operator Name Example Description
&& Logical and x < 10 && x < 15 Returns true only if both the statements are true
|| Logical or x < 15 || x < 10 Returns true if one of the statements is true
! Logical not !(x < 15) Returns true if statement is false.
Example :- 
        
            using System;
            namespace IdleProgrammer
            {{'{'}}
                class Program
                {{'{'}}
                    static void Main(string[] args)
                    {{'{'}}
                        bool result;
                        int x = 10, y = 5;
                        result =((x == y)&&(x>y));
                        Console.WriteLine(result);
                        result =((x == y)||(x>y));
                        Console.WriteLine(result);
                        result = !(x > y);
                        Console.WriteLine(result);
                    {{'}'}}
                {{'}'}}
            {{'}'}}
            Output:
                    False
                    True
                    True

Bitwise Operators

  • Bitwise operatorsare used to perform bit-level operations on Boolean and integer data.
Operator Operator Name Description
& Bitwise AND It will check each bit of first operand with the second operand if both the bits are 1 then the result will be 1 otherwise the result will be 0.
| Bitwise OR It will check each bit of first operand with the second operand if one of them is 1 then the result will be 1 otherwise 0.
^ Bitwise Exclusive OR (XOR) It will check each bit of the first operand with the corresponding bit of its second operand. If one bit is 0 and the other bit is 1, then the result bit will be 1 otherwise the result bit will be 0.
~ Bitwise Complement It will check ieach bit of the operand and make it opposite i.e 1 to 0 and 0 to 1.
<< Bitwise Left Shift It shifts the number to the left based on the specified number of bits.
>> Bitwise Right shift operator It shifts the number to the right based on the specified number of bits
Example :- 
        
            using System;
            namespace Idleprogrammer 
            {{'{'}}
                class Program 
                {{'{'}}
                    static void Main(string[] args) 
                    {{'{'}}
                        int a = 5;            
                        int b = 4;            
                        int c; 
                        c = a & b;        
                        Console.WriteLine(c);
                        c = a | b;            
                        Console.WriteLine(c);
                        c = a ^ b; 
                        Console.WriteLine(c);
                        c = ~a;                
                        Console.WriteLine(c);
                        c = a << 2;     
                        Console.WriteLine(c);
                        c = a >> 2;     
                        Console.WriteLine(c);
                    {{'}'}}
                {{'}'}}
            {{'}'}}
            Output:
                    4                                                                                                                                      
                    5                                                                                                                                      
                    1                                                                                                                                      
                    -6                                                                                                                                     
                    20                                                                                                                                     
                    1

  • Assignment operators are used to assign values to the operand.
Operator Operator Name Example Same As
= Equal to a = 2 a = 2

-= Subtraction Assignment x -= 2 x = x - 2
*= Multiplication Assignment x *= 2 x = x * 2
/= Division Assignment x /= 2 x = x / 2
%= Modulo Assignment x %= 2 x = x % 2
&= Bitwise AND Assignment x &= 2 x = x & 2
|= Bitwise OR Assignment x |= 2 x = x | 2
^= Bitwise Exclusive OR Assignment x ^= 2 x = x ^ 2
>>= Right Shift Assignment x >>= 2 x = x >> 2
<<= Left Shift Assignment x <<= 2 x = x << 2
