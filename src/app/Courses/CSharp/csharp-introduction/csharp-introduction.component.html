Introduction To C#

What is C#?

C# is pronounced "C-Sharp". It is an object-oriented programming language created by Microsoft that runs on the .NET Framework.
C# is often thought of as a hybrid that takes the best of C and C++ to create a truly modernized language. Although the .NET framework supports several other coding languages, C# has quickly become one of the most popular.
C# can be used to create almost anything but is particularly strong at building Windows desktop applications and games.
C# can also be used to develop web applications and has become increasingly popular for mobile development too.
Cross-platform tools such as Xamarin allow apps written in C# to be used on almost any mobile device.

Why Use C#?

  • It is one of the most popular programming language in the world

  • C# is an object oriented language which gives a clear structure to programs and allows code to be reused, lowering development costs.

  • As C# is close to C, C++ and Java, it makes it easy for programmers to switch to C# or vice versa

  • It has a huge community support

How C# programing works?

  • using System; :
    • System is a namespace which contains the commonly used types. It is specified with a using System directive.
  • namespace HelloWorldApp:
    • namespace is just a keyword that is used to define our namespace.HelloWorldApp is the user-defined name given to namespace.
  • class HelloWorld :
    • Here class is just a keyword used to define a class. HelloWorld is the user defined name given to the class. Always keep one thing in mind that you should always start a class name with capital letter that is a good programing practice. Though in visual studio there is a predefined class will be there named as "Program", but if you want to change you can do that
  • static void Main(string[] args) :
    • Here static keyword tells us that this method is accessible without instantiating the class. void keyword tells that this method will not return anything. Main() method is the entry point of our application. In our program, Main() method specifies its behavior with the statement Console.WriteLine(“Hello World!”);.
  • Console.WriteLine("Hello World"); :
    • Here WriteLine() is a method of the Console class defined in the System namespace. WriteLine is used like print method in c that means it is used to show something in the screen. Another thing is if you like to import "using static System.Console;" you need not to write "Console.WriteLine()", you can simply use "WriteLine()" to show something in the screen.
  • Console.ReadKey():
    • This makes the program wait for a key press and prevents the screen from running and closing quickly. This is useful while coding in visual studio.
