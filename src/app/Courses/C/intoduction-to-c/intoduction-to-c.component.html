Introduction to C language

What is c?

C is a programing language developed at AT & T's Bell Laboratories of USA in 1972. It was designed by sir Dennis Ritchie.

Image credit

C programming is a basic programming language that you should learn to proceed further for other programming languages. This language was so popular and was beginning to replace the more familiar languages of that time like PL/I, ALGOL, etc.. This language is so popular because it is reliable,simple and easy to use.

Key Applications where C is used

  • It is used to develop an operating system. There are some popular operating systems are there those are developed using "C" language such as Windows and Apple's OS X, etc.. It is used for developing desktop as well as a mobile phone operating system.
  • It is used for compiler development.
  • 'C' language is widely used in embedded systems.
  • It is widely used in IOT applications.
  • It is used to develop databases. MySQL is the most popular database software which is developed using 'C'.
  • Most of the applications by Adobe are developed using 'C' programming language.
