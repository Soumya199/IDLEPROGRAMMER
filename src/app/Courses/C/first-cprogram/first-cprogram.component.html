Let's deep dive into it

  • As we can see the first line of our program is #include<stdio.h> which is a preprocessor command which tells the compiler to include stdio.h before going for the actual compilation process
  • The Second line of code contains int main(), main is a function from where the actual execution starts, "()" symbol defines that main is a function, before main we have an int that defines return type of method, that means the main method will return an integer type value. if you don't want to return any value then simply use void instead of int.
  • The next line we have printf(), as we have discussed Printf() is a function which is used to print value in console and printf is a C function belonging to the ANSI C standard library, and included in the file stdio.h.
  • The next line contains return 0, As I have already told you that int main() is going to return a value of integer type for that we are returning "0";

Keywords and Identifiers

Keywords are known as a building block of a C program that meaning cannot be changed, There are total 32 keywords in C, and keywords are written in lowercase.

auto double int struct
break else long switch
case enum register typedef
char extern return union
const short float unsigned
continue for signed void
char extern return union
default goto sizeof volatile
do if static while

An identifier is nothing but a name that is assigned to an element in a program. Example, name of a variable, function, etc. As the name sounds, identifiers are used to identify a particular element in a program. Each identifier must have a unique name. Following rules must be followed for identifiers:

  • The first character must be an alphabet or an underscore.
  • It contains only letters, numbers, or underscore.
  • A keyword cannot be used as an identifier.
  • It should not contain any whitespace character.
  • The name must be meaningful.
