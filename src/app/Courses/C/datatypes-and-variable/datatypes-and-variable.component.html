Variable

  • A variable is nothing but just a name given to a storage area.
  • The value of the variable can be changed.
  • Each variable in C has its type which will tell that what type of variable it is..
For creating a variable in C there are certain rules are there please follow these rules while creating a variable.

  1. Each and every variable name should start with alphabets or underscore (_).
  2. Spaces are allowed in variable declaration.
  3. No other special symbols are allowed other than Except underscore (_) in the middle of the variable declaration.
  4. The variable name should not match any keyword.
Example : - int x=10;

Here x is a variable and int specify that x is an integer type variable and 10 is allocated to x.

Data Types

  • As we discussed earlier each variable has a type and that will specify which type of value it will contain.
  • Each data type requires different amounts of memory.

Let's discuss some common data types

  • int : As I have already discussed, int is to specify that the variable is going to hold integer type data.
    Example: int y;
  • char : This data type specifies that the char type variable is going to store a single character value.
    Example: char b='a';
  • float : A float variable can contain both the whole numbers and fractions
    Example: float f=1.7 --> if you will print f you will get 1.7

Format specifiers

  • The format specifiers are used for input and output purposes.
  • By considering the format specifiers compiler get to know that what type of data is in a variable during taking input using the scanf() function and printing using printf() function.
