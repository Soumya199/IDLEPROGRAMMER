Envirnment SetUp

As we are going to set up the our envirnment to execute C programming we should have some basic knowledge about the editors and compiler.

Text Editors

Text editors are generaly used to type our code and save them as per their extensions. Text editors can vary depending upon your OS, We can say that Notepad is a product from Microsoft so that can be used in Windows only, like that you can choose your editor based on your OS.
ex- Notepad,Notepad++,VSCode etc

C Compiler

As we all know computer only understand 0 & 1 so what ever we have written in the editor that needs to be be "compiled", into machine language so that your CPU can actually execute the program as per the instructions given. Keep in mind that what ever we write in the editor needs to follow the syntax of that programming language otherwise that is going to give you error messages.

C Installation

There are lots of compilers are available to execute c and c++, So only thing you need to do is to download one of them and install it in your System and you are ready to go.

To download Turbo c++ you can just type "download turbo c++" on google and you will find lots of website to download, just download and install the exe file. The installation proccess is so simple like clicking next-next button.

Another way is just go and download the Visual studio code after download just install this.
After installation just open it by double cliking it.
Click on the extension button that can be found on the left side.
In the extension search box type "C/C++" and press enter
You can find a extension there from Microsoft just clik install
Now create a new file using comand "control+n" for Windows and "commad+n" for mac and than press "control+s" for windows and "command+s" for mac to save the file with a file name and the extension should be .c for C or .cpp for C++
ex: filename.c/filename.cpp

Now you are ready to code in c.
