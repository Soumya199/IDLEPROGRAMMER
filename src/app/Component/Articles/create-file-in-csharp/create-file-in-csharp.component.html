How to create a text file in memory using C#

Hi readers, hope you are doing good. Today in this article I will explain you how to create a text file in memory using C# programing language because I got this question from 10-15 people. So let's get started....

As we all know learning C# is simpler than other languages. C# provides the facility to create a text file in memory and use it as per your requirement. For example Let's consider we have a console application. As we all know if we use array or collections to store data that will be available for us till the program is in running. But Csharp provide us a facility to hold the data in memory even after execution is finished. That is known as Serialization and Deserialization

In a simple way we can say that Serialization is known as to write something to the file in memory and Deserialization is known as to read data from the file. But actually in Serialization we are just saving an object to the file and that will need a formatter to do all these so we are not going to discuss more about Serialization and Deserialization now. You can find all about this in C# course section.Click here for full C# coruse for free.

So let's begin the process to create a simple text file using C#

Steps
  • Open your visual studio, In my case I have Visual studio 2019.
  • Click on create a new Project
  • In the search box search for console application, you can find a C# console application (.Net Framework) click on that and then click Next
  • Give a name to your project and choose a location if you want and then click on Create
  • import the namespace using System.IO;
  • With the main method write the below code
    • string path = @"D:\Tkss\Result.txt";
    • Stream stream = File.Open(path, FileMode.OpenOrCreate);
  • In the path just put your path where you want to have your txt file and the txt file name, This will create a txt file in your memory

Let's discuss how this code creating a text file..

Here in the first line we have declared a path of string type, path is only going to hold the path where we are going to create a txt file along with the file name here i have "Result.tx"."Result" is the file name and ".txt" is the file extension.

In the second line, we have declared Strem of Stream type. Basically Stream is a class and that is used to read or write data to or from a file. The stream is defined in the System. Io package so we have to import that to take benefit of it. To know more about Stream just select Stream right click on it and click on go to defination. You can find everything about that. We have File.Open is there this is writen because we are going to deal with file so we have used File that is also a class and Open is a method. for more you can go to it's defination and explore everything.As a parameter we have given path and the file mode , there are different file modes are there that you can use but i have used "OpenOrCreate" because that is going to check the availbility of the txt file if the file exist that will not create another one otherwise it will create a file with the name that you have given.
